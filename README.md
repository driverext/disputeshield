# DisputeShield

DisputeShield is a zero-budget SaaS that generates bank-ready chargeback evidence packets (PDF + ZIP) directly in the browser. It helps merchants compile timelines, policies, and attachments into a clear, structured packet without uploading files to a server.

Live: disputeshield.app

## Environment Variables

- `TURNSTILE_SECRET` (Cloudflare Pages server env)
- `VITE_TURNSTILE_SITEKEY` (client)
